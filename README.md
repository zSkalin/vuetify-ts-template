# Vuetify TypeScript Template

## Packages

* VueJS
* Vuetify
* axios
* vue-router
* vuex
* vue-class-component
* vue-property-decorator
* vuex-module-decorators

## Usage
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
