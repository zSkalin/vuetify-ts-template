# Vuetify TypeScript Template

## Packages

* VueJS - V2.6
* Vuetify - V2.3.13
* axios - V0.20.0
* vue-router - V3.4.6
* vuex -V3.5.1
* vue-class-component - V7.2.6
* vue-property-decorator - V9.0.2

## Usage
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
